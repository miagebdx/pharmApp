Download

  1. git clone https://github.com/miagebdx/pharmApp.git
  2. make all
  3. cd Build && ./mainTerm

Gui Version

Warning : Need GTK & GTKmm library v3.0

  1. git checkout dev-gui
  2. make mainGui
  3. cd Build && ./mainGui

The binary file are located on the build folder.
